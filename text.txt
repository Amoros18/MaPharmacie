Voici un exemple de code HTML et CSS pour créer un dashboard avec un en-tête, une barre de navigation latérale gauche et un contenu à droite :

HTML :

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Dashboard</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <header>
    <h1>Dashboard</h1>
  </header>

  <div class="container">
    <nav>
      <ul>
        <li><a href="#">Home</a></li>
        <li><a href="#">Reports</a></li>
        <li><a href="#">Settings</a></li>
      </ul>
    </nav>

    <main>
      <h2>Welcome to the Dashboard</h2>
      <p>This is the main content area of the dashboard.</p>
    </main>
  </div>
</body>
</html>
```

CSS :

```css
/* Global Styles */
body {
  margin: 0;
  font-family: Arial, sans-serif;
}

/* Header Styles */
header {
  background-color: #333;
  color: #fff;
  padding: 1rem;
  text-align: center;
}

/* Container Styles */
.container {
  display: flex;
  height: calc(100vh - 4rem); /* Subtract header height */
}

/* Navigation Styles */
nav {
  background-color: #f2f2f2;
  border-right: 1px solid #ddd;
  padding: 1rem;
}

nav ul {
  list-style-type: none;
  margin: 0;
  padding: 0;
}

nav li {
  margin-bottom: 0.5rem;
}

nav a {
  color: #333;
  text-decoration: none;
}

nav a:hover {
  color: #000;
}

/* Main Content Styles */
main {
  flex-grow: 1;
  padding: 1rem;
}
```

Explication :

1. Le HTML crée la structure de base avec un en-tête, une barre de navigation latérale gauche et une zone de contenu principale.
2. Le CSS applique les styles pour la mise en page et l'apparence du dashboard.
3. La classe `container` crée un conteneur flexbox qui permet de positionner la barre de navigation à gauche et le contenu principal à droite.
4. La barre de navigation `nav` a un fond de couleur gris clair et une bordure à droite.
5. Le contenu principal `main` occupe le reste de l'espace disponible et est stylisé avec un remplissage.

Vous pouvez personnaliser les styles et ajouter davantage de contenu et de fonctionnalités selon vos besoins.
